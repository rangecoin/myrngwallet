FAQ

Q: Are my private keys secured?
A: Yes they are secured, your private keys never leave your browser.

Q: How is this secure? Its on a web browser!
A: This wallet has replicated the core features of Rangecoin daemon in JavaScript! So the process of creating and signing the transactions are all done within the browser. No sensitive information is sent through the network.

Guide

1. Creating a new wallet

  • Go to https://rangecoin.github.io/myrngwallet/
  • Enter a strong but easy to remember secret phrase. Save this somewhere and do NOT forget it.
  • Click the Generate Wallet Button
    • This creates a deterministic wallet based on your secret phrase.

2. Accessing an existing wallet

  • If you've created a wallet on https://rangecoin.github.io/myrngwallet/ via the secret phrase, simply re-enter the same phrase.
  • For those with a wallet.dat file generated by GUI wallets. Click settings (), select the Load wallet.dat option and load your wallet.dat file
  • To unlock a single address, Click settings (), select the Paste private key option, paste in your private key and click the unlock button
